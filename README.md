# Houston Bloom

**Equip communities with the technology to co-create and collaborate.**

Houston Bloom is a 2027 field program built on the [Community Legibility Spec](#the-community-legibility-spec). It gives a community the tools to find what it needs in itself, build on each other's work, and own the data it makes together.

## The Community Legibility Spec

The program runs on an open specification, published as two papers:

- [Dual-Ledger Authentication](https://doi.org/10.6084/m9.figshare.32576631) · how two paired records authenticate a real activity without a central authority
- [Local Assembly](https://doi.org/10.6084/m9.figshare.32772147) · how a community assembles its own legible, portable record

## The site
